# Agent Card Editor

> 可视化 A2A Agent Card 编辑器 — 实时 JSON 预览、验证、导入导出

[![GitHub](https://img.shields.io/badge/GitHub-100000?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Erich956389473/agent-card-editor)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ 功能特性

- **可视化编辑** — 图形界面编辑 Agent Card
- **实时预览** — JSON 实时同步更新
- **Schema 验证** — 自动验证 A2A 规范
- **导入导出** — 支持 JSON 文件导入导出

## 🚀 快速开始

### 安装

`ash
git clone https://github.com/Erich956389473/agent-card-editor.git
cd agent-card-editor
npm install
`

### 使用

`ash
# 启动开发服务器
npm start

# 构建生产版本
npm run build
`

## 📖 使用说明

1. 打开编辑器界面
2. 填写 Agent Card 信息（名称、描述、技能等）
3. 实时预览 JSON 输出
4. 导出为 JSON 文件
